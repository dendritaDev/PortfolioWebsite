# CV page

This folder contains the web CV for Oriol Martin Corella.

## Open locally

Open `cv/index.html` in a browser, or navigate to `cv/` from the portfolio homepage when served through GitHub Pages.

## Export to PDF from Chrome

1. Open `cv/index.html` in Chrome.
2. Click `Print / Save PDF`, or press `Ctrl+P`.
3. Choose `Save to PDF`.
4. Use these print settings:
   - Paper size: `A4`
   - Background graphics: enabled
   - Margins: default, or none if you want Chrome to rely only on the page layout
   - Scale: default/100%; reduce slightly only if Chrome creates a second page

The page includes `@page` and `@media print` CSS so the navigation and action buttons are hidden in the exported PDF.
